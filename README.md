# PND26 Package Index

A clean static frontend project created for Project Nationdex 2026 in order to list extras.

## Submitting extras

In order to submit an extra, just reach to `extras.json` and add your entry.

```json
{
    "id": "berklikesfemboys",
    "repo": "https://github.com/Berk404/berklikesfemboys",
    "branch": "main"
}
```

Make sure to replace the silly `id`, `repo`, and `branch` inputs with yours.
